Overview

Timeline • Deliverables • Status

Focus Flow is a minimalist Android task manager focused on reducing cognitive load and promoting deep work. The MVP focuses on three core areas: daily planning (Today), future planning (Upcoming) and management tools (Browse + Focus timer).

Project Goals

  • Design an intuitive, minimal UX for daily task flow
  • Speed up task creation with contextual inputs
  • Introduce a focus timer tied to tasks to measure time spent
  • Create a compact visual identity aligned with calm productivity

Research & Insights

Conducted comparative analysis of Todoist, TickTick, and Google Tasks. Found users wanted fewer features in the main view and faster ways to add focused items.

UX Flow & Screens

Key screens, interactions and task creation flow

Primary flow

  1. Launch → Splash → Login (Email/Google/Facebook)
  2. Home → Today view shows tasks for the day
  3. User taps + to add a task → chooses date, priority, reminder and category
  4. Start Focus timer for a task to record time spent
  5. Completed tasks move to Completed list in Browse

Prototype & Usability Testing

Built an interactive Figma prototype covering task creation, calendar selection and the focus timer. Performed early usability testing with 6 participants to validate task creation speed and visibility of priority states.

Learnings & Next Steps

What came next

Key learnings

  • Keep primary screens minimal — avoid presenting too many controls at once.
  • Users loved the quick add flow and focus timer as motivational feedback.
  • Providing a lightweight categorization system made tasks feel more manageable.

Next steps

  • Ship an MVP Android build with cloud sync and user accounts.
  • Iterate on smart suggestions for scheduling based on past behavior.
  • Add lightweight analytics to measure focus sessions and task completion rates.
