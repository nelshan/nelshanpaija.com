Pong – 2D Arcade Game Clone

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

This project is a faithful recreation of the classic Pong game, developed to demonstrate proficiency in Unity and fundamental game development principles. The game features two-player local multiplayer, responsive paddle controls, realistic ball physics, and a scoring system where the first player to reach 10 points wins.
