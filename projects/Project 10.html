Skullgun – 2D Top-Down Shooter

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

A simple top-down shooter where you must survive endless waves of enemies. Armed with auto-firing projectiles, you'll move, and eliminate foes while trying to stay alive. Each wave increases the number of enemies. Just a simple mechanics, quick gameplay, and non-stop action—perfect for a short session.

Key Features

  • Auto-Targeting Combat: Implemented an auto-fire system that targets the nearest enemy, streamlining combat mechanics.
  • Wave-Based Enemy Spawning: Designed escalating enemy waves to progressively challenge the player.
  • Responsive Controls: Utilized Unity's input system to ensure smooth and intuitive player movement.
  • User Interface: Developed a clean UI displaying wave count and game over prompts, enhancing user experience.

Controls

  • Move: WASD or Arrow Keys
  • Auto-Shoot: Automatically targets and fires at the nearest enemy

Development Highlights

  • Solo Development: Managed all aspects of the project, including game design, programming, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls.
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

Developing Skullgun provided valuable experience in creating engaging gameplay loops and refining user interactions. The project honed skills in 2D game development, AI behavior scripting, and user interface design.
