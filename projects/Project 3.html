Flappy Bird – 2D Arcade Game Clone

Role: Solo Developer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Floppy Bird is a faithful recreation of the classic Flappy Bird game, developed using Unity and C#. This project replicates the simple yet challenging mechanics of the original, where players control a bird navigating through a series of pipes without collision. The game features responsive controls, dynamic obstacle generation, and a scoring system to track player progress.​

Key Features

  • Authentic Gameplay Mechanics: Implemented gravity-based movement and collision detection to mirror the original game's feel.
  • Dynamic Obstacle Generation: Pipes are procedurally generated with varying gaps to increase difficulty and replayability.
  • Score Tracking: scoring system that updates as players successfully navigate through obstacles.
  • User Interface: Clean and minimalist UI displaying current and high scores

Controls

  • Flap: Spacebar or Left Mouse Click
  • Pause: Esc key

Development Highlights

  • Solo Development: Handled all aspects of development, including programming, design, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls.
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

This project served as an excellent exercise in understanding the core mechanics of game development. Recreating a classic game like Flappy Bird allowed for a focus on refining gameplay elements and ensuring a polished user experience.
