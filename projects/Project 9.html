Dino Runner – 2D Endless Runner Game

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Dino Runner is a 2D endless runner game inspired by the classic offline dinosaur game found in Google Chrome. Developed using Unity and C#, this project replicates the engaging mechanics of the original, challenging players to navigate a pixel-art dinosaur through a treacherous prehistoric landscape filled with obstacles.

Key Features

  • Responsive Controls Implemented precise input handling to ensure smooth jumping and crouching mechanics.
  • Dynamic Obstacle Generation: Utilized procedural generation to create a variety of obstacles, enhancing replayability.
  • Scoring System: Developed a real-time score tracking system to monitor player progress and encourage improvement.
  • Optimized Performance: Ensured seamless gameplay across web platforms through efficient coding practices.

Controls

  • Jump: Press Spacebar
  • Crouch: Hold Left Shift

Development Highlights

  • Solo Development: Managed all aspects of the project, including game design, programming, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls.
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

Recreating Dino Runner provided valuable experience in understanding classic game mechanics and translating them into modern development environments. The project honed skills in 2D game development, collision detection, and user input management.
