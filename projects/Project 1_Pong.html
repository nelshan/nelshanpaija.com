Pong – 2D Arcade Game Clone

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

This project is a faithful recreation of the classic Pong game, developed to demonstrate proficiency in Unity and fundamental game development principles. The game features two-player local multiplayer, responsive paddle controls, realistic ball physics, and a scoring system where the first player to reach 10 points wins.

Key Features

  • Two-Player Local Multiplayer: Engage in head-to-head matches with intuitive controls.
  • Realistic Ball Physics: Implemented physics-based ball movement and collision detection for an authentic gameplay experience.
  • Score Tracking: Integrated a scoring system that displays each player's score, culminating in a win condition at 10 points.
  • User Interface: Designed a clean and minimalist UI to reflect the classic arcade aesthetic.​

Controls

  • Player 1 (Left Paddle): W – Move Up | S – Move Down
  • Player 2 (Right Paddle): Up Arrow – Move Up | Down Arrow – Move Down​

Development Highlights

  • Solo Development: Handled all aspects of development, including programming, design, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls.
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

This project served as an excellent exercise in understanding the core mechanics of game development. Recreating a classic game like Pong allowed for a focus on refining gameplay elements and ensuring a polished user experience.
