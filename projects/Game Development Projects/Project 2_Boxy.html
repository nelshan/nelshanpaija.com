BOXY – 2D Top-Down Shooter

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

BOXY is a 2D top-down shooter made using Unity engine and C# scripting. The game features four distinct gun types, diverse enemies with increasing difficulty, and intense wave-based combat. I implemented dynamic enemy behavior and challenging mini-boss encounters.

Key Features

  • Four Gun Types: Each weapon offers unique gameplay mechanics and strategic advantages.
  • Wave-Based Combat: Progressive difficulty with increasingly challenging enemy waves.
  • Dynamic Enemy Behavior: AI-driven enemies that adapt to player actions.
  • Mini-Boss Encounters: Special boss battles that test player skills.

Controls

  • Movement: WASD or Arrow Keys
  • Shoot: Mouse Click
  • Switch Weapons: Number Keys 1-4

Development Highlights

  • Unity 2D Development: Leveraged Unity's 2D capabilities for smooth gameplay.
  • C# Scripting: Implemented complex game mechanics and enemy AI.
  • Game Balance: Carefully tuned difficulty progression and weapon systems.

Reflection

This project demonstrated my ability to create engaging combat systems and implement AI-driven enemies. The wave-based progression system provides a satisfying challenge that keeps players engaged.
