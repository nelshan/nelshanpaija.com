Conway's Game of Life – Cellular Automaton

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Project On: GitHub

Overview

A simple implementation of Conway's Game of Life in Unity. This cellular automaton devised by John Conway in 1970, simulates the evolution of cells based on simple rules, resulting in complex patterns. Conway's Game of Life is a well-known example of a zero-player game, meaning that once initialized, the system evolves autonomously.

Key Features

  • Classic Rules: Implements the original Conway's Game of Life rules.
  • Interactive Grid: Click to place or remove cells before starting.
  • Autonomous Evolution: Watch as patterns emerge and evolve automatically.
  • Speed Control: Adjust simulation speed to observe patterns in detail.

Controls

  • Place/Remove Cells: Left Click on grid
  • Start/Stop: Spacebar
  • Clear Grid: C key
  • Speed Control: Up/Down Arrow keys

Development Highlights

  • Algorithm Implementation: Coded the classic Game of Life rules in C#.
  • Grid System: Created an efficient 2D grid for cell management.
  • Performance Optimization: Implemented efficient cell state calculations.

Reflection

This project demonstrated the beauty of simple rules creating complex emergent behavior. It's fascinating how basic mathematical rules can generate such intricate and unpredictable patterns.
