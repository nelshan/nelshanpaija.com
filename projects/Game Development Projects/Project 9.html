Dino Runner – Endless Runner Game

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Dino Runner is an endless runner game inspired by Chrome's dinosaur game, developed using Unity and C#. Players control a dinosaur character that must jump over obstacles and avoid enemies in an endless desert landscape.

Key Features

  • Endless Gameplay: Infinite procedurally generated obstacles and challenges.
  • Simple Controls: One-button jumping mechanics for easy accessibility.
  • Progressive Difficulty: Increasing speed and complexity as the game progresses.
  • Score Tracking: Distance-based scoring system to encourage replayability.

Controls

  • Jump: Spacebar or Mouse Click
  • Objective: Run as far as possible while avoiding obstacles

Development Highlights

  • Procedural Generation: Created dynamic obstacle spawning system.
  • Physics Implementation: Realistic jumping mechanics and collision detection.
  • Performance Optimization: Efficient rendering for smooth endless gameplay.

Reflection

This project demonstrated the appeal of simple, addictive gameplay mechanics. The endless runner format provides a perfect balance of challenge and accessibility that keeps players engaged.
