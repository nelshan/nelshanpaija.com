Frogger – Classic Arcade Game

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

This project is a faithful recreation of the classic 1981 arcade game Frogger, developed using Unity and C#. The game challenges players to guide a frog across a busy road and a treacherous river, aiming to reach designated safe zones while avoiding various obstacles.

Key Features

  • Classic Gameplay: Faithful recreation of the original Frogger mechanics.
  • Two-Phase Challenge: Navigate both road and river sections.
  • Progressive Difficulty: Increasing speed and complexity of obstacles.
  • Score System: Track progress and compete for high scores.

Controls

  • Movement: Arrow Keys or WASD
  • Objective: Guide the frog to safe zones
  • Challenge: Avoid cars and navigate river currents

Development Highlights

  • Unity 2D Development: Implemented classic arcade mechanics.
  • Collision Detection: Precise collision handling for gameplay.
  • Level Design: Created authentic arcade-style progression.

Reflection

Recreating this classic arcade game was a great exercise in understanding fundamental game mechanics. The simple yet challenging gameplay demonstrates the timeless appeal of well-designed arcade games.
