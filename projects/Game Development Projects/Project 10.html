Skullgun – Top-Down Shooter

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Skullgun is a simple top-down shooter where you must survive endless waves of enemies. Armed with auto-firing projectiles, you'll move and eliminate foes while trying to stay alive. Each wave increases the number of enemies. Just simple mechanics, quick gameplay, and non-stop action—perfect for a short session.

Key Features

  • Auto-Firing Combat: Continuous projectile system for intense action.
  • Wave-Based Progression: Increasingly challenging enemy waves.
  • Simple Controls: Easy-to-learn movement and combat mechanics.
  • Quick Sessions: Perfect for short, intense gaming sessions.

Controls

  • Movement: WASD or Arrow Keys
  • Shooting: Automatic - just move to aim
  • Objective: Survive as many waves as possible

Development Highlights

  • Auto-Targeting System: Implemented automatic shooting mechanics.
  • Wave Management: Created dynamic enemy spawning system.
  • Performance Optimization: Efficient rendering for smooth action gameplay.

Reflection

This project focused on creating immediate, satisfying gameplay with minimal complexity. The auto-firing mechanic allows players to focus on movement and positioning rather than aiming.
