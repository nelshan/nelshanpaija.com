Block Breaker – Classic Arcade Game

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Block Breaker is a 2D arcade game developed using Unity and C#, inspired by the classic brick-breaker genre. Players control a paddle to bounce a ball and break all the blocks on the screen.

Key Features

  • Classic Brick-Breaker Gameplay: Bounce ball with paddle to break blocks.
  • Physics-Based Ball Movement: Realistic ball physics and collision detection.
  • Level Progression: Multiple levels with increasing difficulty.
  • Power-ups: Various power-ups to enhance gameplay.

Controls

  • Paddle Movement: Left/Right Arrow Keys or A/D
  • Launch Ball: Spacebar
  • Objective: Break all blocks to complete the level

Development Highlights

  • Physics Implementation: Realistic ball and paddle physics.
  • Level Design: Created engaging block arrangements.
  • Power-up System: Implemented various gameplay enhancements.

Reflection

This project demonstrated the importance of satisfying physics and responsive controls in arcade games. The brick-breaker genre remains engaging due to its simple yet challenging mechanics.
