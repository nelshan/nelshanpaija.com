Frogger – 2D Arcade Game Clone

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

This project is a faithful recreation of the classic arcade game Frogger, developed using Unity and C#. Players navigate a frog across busy roads and hazardous rivers, aiming to reach the safety of the opposite bank. The game emphasizes timing, precision, and quick reflexes, capturing the essence of the original while providing a smooth and responsive experience.

Key Features

  • Authentic Gameplay Mechanics: Implemented classic movement and obstacle patterns to mirror the original game's challenge.
  • Responsive Controls: Utilized Unity's input system to ensure precise and intuitive player movement.
  • Dynamic Obstacle Generation: Vehicles and logs are procedurally generated with varying speeds and patterns to increase difficulty and replayability.
  • Scoring System: Real-time score tracking to monitor player progress and encourage improvement.

Controls

  • Move Up: W key
  • Move Down: S key
  • Move Left: A key
  • Move Right: D key

Development Highlights

  • Solo Development: Managed all aspects of the project, including game design, programming, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

This project has provided valuable experience in understanding classic game mechanics and translating them into modern development environments. The project honed skills in 2D game development, collision detection, and user input management.
