Run from Unknown – 3D First-Person Survival Horror Game

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Download)

Play it on: Itch.io

Overview

Run from Unknown is a short, first-person survival horror game that immerses players in a tense and mysterious environment. Waking up alone in a dark, foggy building with no memory of how they got there, players must navigate through eerie corridors and rooms while being pursued by a terrifying creature. The game emphasizes atmosphere and suspense, offering a chilling experience without combat—survival depends solely on the player's ability to evade and escape.

Key Features

  • Defenseless Horror Mechanics: Players cannot fight back, heightening tension through stealth and evasion.
  • Puzzle Integration: Environmental puzzles that unlock new areas and advance the narrative.
  • Atmospheric Sound Design: Immersive audio elements, including ambient noises and jump scares, to enhance the horror experience.
  • AI-Driven Monster: A relentless creature utilizing Unity’s NavMesh Agent for intelligent pathfinding and player detection.​

Development Highlights

  • Solo Development: Managed all aspects of the project, including game design, programming, and asset integration.
  • Tool Utilization: Employed Unity for game development, Blender for 3D modeling, PlayFab for user authentication, and Audacity for sound editing.
  • Optimized Performance: Ensured smooth gameplay and responsive controls through efficient coding practices.​

Reflection

This project served as a comprehensive exercise in creating an engaging horror experience. It provided valuable insights into atmospheric design, AI behavior, and the integration of various tools to build a cohesive game.
