Block Breaker – 2D Arcade Game Clone

Role: Solo Developer | Designer | Programmer

Engine: Unity

Platform: PC (Web)

Play it on: Itch.io

Overview

Block Breaker is a 2D arcade game developed using Unity and C#, inspired by the classic brick-breaker genre. Players control a paddle to bounce a ball and break all the blocks on the screen. The game challenges players' reflexes and precision, offering endless gameplay as blocks respawn upon clearance.

Key Features

  • Classic Gameplay Mechanics: Implemented traditional brick-breaking mechanics, ensuring an authentic arcade experience.
  • Responsive Controls: Utilized Unity's input system to provide smooth and intuitive paddle movement.
  • Endless Replayability: Designed the game to respawn blocks upon clearance, allowing for continuous play and high-score pursuits.
  • User Interface: Developed a clean UI displaying scores and lives, enhancing user engagement.

Controls

  • Move Left: A Key
  • Move Right: D Key
  • Ball Launch: Spacebar key

Development Highlights

  • Solo Development Managed all aspects of the project, including game design, programming, and testing.
  • Unity Engine Utilization: Leveraged Unity's 2D capabilities to create smooth gameplay mechanics and responsive controls.
  • Code Optimization: Wrote clean and efficient C# scripts to manage game logic, input handling, and collision responses.

Reflection

Developing Block Breaker provided valuable experience in recreating classic game mechanics and refining user interactions. The project honed skills in 2D game development, collision detection, and user interface design.
